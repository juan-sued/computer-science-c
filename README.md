## Como usar

1. Clone o repositório:
  ```bash
  git clone https://github.com/seu-usuario/computer-science-c.git
  ```
2. Navegue até o diretório do projeto desejado:
  ```bash
  cd nome-do-projeto
  ```
3. Siga as instruções específicas de cada projeto para executar.
